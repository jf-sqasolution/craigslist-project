#Craigslist Automation Project
##Search for jobs on Craigslist based on keywords

*Project for an automation test using Selenium WebDriver to search for a set of keywords on Craigslist.*

**Java 7**

* Maven
* Log4j
* TestNG
* Selenium WebDriver
* Apache POI
* JDBC
* MySQL
